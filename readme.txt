Voici un résumé des étapes à suivre pour télécharger et exécuter le programme ProjetLangageNaturel :
	1-Ouvrir une ligne de commande et exécuter la commande suivante pour cloner le projet à partir de GitHub :  git clone https://github.com/bachir-hadjoudja/Projet-Langage-Naturel.git
	2-Se déplacer dans le répertoire racine du projet et exécuter la commande suivante pour installer toutes les bibliothèques requises : pip install -r bibliotheque.txt
	3-Exécutez le programme en ouvrant le fichier main.py dans votre IDE Python préféré et en exécutant le code à partir de celui-ci : python3 main.py
	4-Une fois le programme lancé, suivez les instructions qui s'affichent à l'écran pour entrer les inputs nécessaires.
